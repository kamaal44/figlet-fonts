# Figlet-fonts
Installing font for figlet packages

Installations
````
$ git clone https://github.com/saydog/figlet-fonts.git
$ cd figlet-fonts
$ chmod +x termux-install.sh
$ ./termux-install.sh
````
Usage
````
$ apt-get install figlet
$ figlet -f "Namefonts" YourText
````
Show list font name
````
$ figfonts
````
